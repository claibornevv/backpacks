# Backpacks Plugin

### A simple plugin for adding a backpack functionality to the game through the use of a server plugin

---

## How to use the plugin

Current commands:
1. /backpack [reset|upgrade] <player> (no arguments to access backpack)
2. /seebackpack (Admin command to access the backpack of others)
3. /setbackpacksize <player\> <size\>
4. /setbackpacksize <size\>
5. /setdefaultbackpacksize <size\>
6. /balance

## Permissions

1. backpacks.backpack
2. backpacks.seebackpack
3. backpacks.setdefaultbackpacksize
4. backpacks.setbackpacksize
5. backpacks.balance
6. backpacks.reset.others
